Welcome to the Dashboard

This page demonstrates a two-column layout using the Carbon UI Shell with a fixed side navigation (3 columns) and a main content area (13 columns). The grid adapts responsively across breakpoints, with the navigation collapsing on smaller screens.

Grid Layout Pattern

The side navigation occupies exactly 3 grid columns on large viewports (lg breakpoint). The content area takes the remaining 13 columns, creating a balanced application layout. This pattern is commonly used for data-rich applications and dashboards.

Responsive Behavior

At medium and small breakpoints, the layout adapts to vertical stacking. The Carbon UI Shell components handle the responsive behavior automatically, with the side navigation becoming collapsible or fully stacked depending on the viewport size.

Implementation Notes

Grid classes are placed directly on the content containers (no wrapper divs). The UI Shell components are imported via the index import for optimal code-splitting. All spacing and typography use Carbon design tokens to maintain visual consistency.
